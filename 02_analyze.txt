Знакомство с веб-технологиями (семинары)
Домашнее задание 1
2. Создайте файл analyze.txt, в котором проанализируйте 
структуру страницы сайта https://dzen.ru/ 
(опишите коротко, своими словами), а именно:
    * есть ли шапка сайта;
    * есть ли подвал сайта;
    * как и где расположен контент, есть ли особенности у контента;
    * есть ли дополнительные элементы на странице (виджеты).

Результаты выполнения задания:
2.1. Есть ли шапка сайта? — ЕСТЬ (условно)
Шапка сайта по сути "пустая". 
В традиционном понимании к шапке можно отнести лого и название "Дзен".
Вместо нее есть динамеческая панель управления (вертикальная и горизонтальная).

2.2. Есть ли подвал сайта? — ЕСТЬ (условно)
Footer есть в левом нижнем углу. Сделан "для галочки", поскольку
постоянно закрыт навязчивым окном с предложением установить "Яндекс".

2.3. Как и где расположен контент, есть ли особенности у контента?
Тело контента (body) размещено в 2 столбца по центру. Неудобно для пользователя,
поскольку значительная чать экрана остается свободной, текст получается
либо очень мелкий, либо пользователь получет слишком мало информации
и вынужден прокручивать экран.

2.4. Есть ли дополнительные элементы на странице (виджеты)? — ЕСТЬ
Виджеты можно вызвать кнопками меню или явно установить на экране
в результате настройки. По умолчанию в правом верхнем углу 
расположен виджет с прогнозом погоды.
При изменении масштаба экрана в левой части появляется вертикальное 
меню с возможностью вывода на экран дополнительных опций.
